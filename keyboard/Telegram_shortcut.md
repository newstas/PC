# Горячие клавиши для Телеграм

Выдешяем текст и нажимает сочетание клавишь

### Для windows:

`Ctrl + B` = **Жирный**

`Ctrl + I` = *Курсив*

`Ctrl + U` = Подчеркнутый
    
`Ctrl + shift + X` = Перечеркнутый
    
`Ctrl + shift + M` = Моноширинный (код)
    
`Ctrl + shift + N` = Очистить стили


`Cmd + B` = **Жирный**

```

__курсив__
`код`
```
